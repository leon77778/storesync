# StoreSync — eCommerce Order Import Pipeline

A Laravel application built for magic42's technical interview assignment,
demonstrating agentic coding practices with Claude Code.

## What it does
- Upload a CSV file of eCommerce orders
- Each order is dispatched as a queued background job
- Jobs validate data, calculate totals, and send confirmation emails
- Live dashboard shows job status (pending/processing/completed/failed)

## Tech Stack
- PHP 8.2 / Laravel 11
- MySQL
- Redis + Laravel Horizon
- Tailwind CSS

## AI Tool Used
Claude Code (Anthropic) — VS Code extension
